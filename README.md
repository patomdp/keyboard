# Arch — Mechanical Keyboards

Premium mechanical keyboard e-commerce website built with React, TailwindCSS, and Framer Motion.

## Tech Stack

- React 19
- TailwindCSS v3.4
- Framer Motion
- Phosphor Icons
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
├── src/
│   ├── App.jsx         # Main application
│   ├── main.jsx        # Entry point
│   └── index.css       # Global styles
├── public/
│   └── keyboard.svg    # Favicon
├── index.html
├── package.json

├── vite.config.js
└── postcss.config.js
```
